## ⚠️ 현재 문제점

### Etherscan API "NOTOK" 오류
- API 키가 설정되어 있지만 "NOTOK" 응답을 받고 있음
- 가능한 원인:
  1. **API 키 만료 또는 무효화**
  2. **Rate limit 초과** (일일 100,000 calls 제한)
  3. **API 키 권한 문제**

---

## 🔍 해결 방법

### 1. API 키 재확인
1. Etherscan 대시보드 접속: https://etherscan.io/myapikey
2. API 키 상태 확인
3. 필요시 새 API 키 생성

### 2. Rate Limit 확인
- 무료 플랜: 5 calls/second, 100,000 calls/day
- 현재까지 사용량 확인 필요

### 3. API 키 재설정
```bash
# .env 파일에 새 API 키 추가
ETHERSCAN_API_KEY=your_new_api_key_here
```
