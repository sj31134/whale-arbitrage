# Project 2 구현 계획

## 목적
**업비트(KRW), 바이낸스(USDT), 비트겟(USDT) 간의 시세 괴리(김치 프리미엄)를 분석하여 통계적 차익거래 기회를 백테스트**

## 핵심 가설
프리미엄이 평균 회귀하는 특성을 활용하여 Z-Score 기반 진입/청산 신호 생성

## 구현 단계

### 1단계: 백테스트 엔진 개선
- [x] 3개 거래소 지원
- [ ] Look-ahead Bias 제거 (첫 30일 명시적 제외)
- [ ] 리스크 관리 로직 추가 (손절매, 최대 보유 기간)
- [ ] 일별 자본 곡선 생성

### 2단계: 성과 지표 개선
- [ ] 연율화 Sharpe Ratio 계산
- [ ] 일별 MDD 계산
- [ ] 벤치마크 비교 (Buy & Hold)

### 3단계: 실행 가능성 검증
- [ ] 선물 시장 활용 방안 검토
- [ ] 실제 거래 비용 시뮬레이션

### 4단계: 최종 리포트 생성
- [ ] 상세 성과 리포트
- [ ] 거래 내역 분석
