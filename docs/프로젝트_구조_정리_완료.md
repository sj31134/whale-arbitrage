**정리 후**:
```
docs/
```

## 🔍 주요 디렉토리 구조

### app/
Streamlit 웹 애플리케이션
- `main.py`: 메인 앱 진입점
- `pages/`: 11개 페이지 모듈
- `utils/`: 9개 유틸리티 모듈

### scripts/
데이터 수집, 분석, 유지보수 스크립트
- `collectors/`: 데이터 수집 스크립트
- `analysis/`: 데이터 분석 스크립트
- `maintenance/`: 유지보수 스크립트
- `main/`: 메인 실행 스크립트
- `subprojects/`: 서브 프로젝트 전용 스크립트
  - `arbitrage/`: Project 2 (차익거래)
  - `risk_ai/`: Project 3 (리스크 AI)

### docs/
프로젝트 문서 (카테고리별 정리)
- `deployment/`: 배포 가이드
- `guides/`: 사용 가이드
- `reports/`: 프로젝트 보고서
- `analysis/`: 데이터 분석 문서

### trading_bot/
자동매매 봇 모듈 (독립적)
- 기존 프로젝트와 완전히 분리
- 기존 데이터는 읽기 전용으로 접근
