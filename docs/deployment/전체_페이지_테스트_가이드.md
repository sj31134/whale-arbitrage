## 1. 차익거래 비용 계산기

### 테스트 절차
1. 메뉴에서 "📊 차익거래 비용 계산기" 선택
2. 사이드바에서 기간 선택 (예: 2024-01-01 ~ 2024-01-31)
3. 거래소 선택 (예: 업비트-바이낸스)
4. "🔍 계산 실행" 버튼 클릭

### 확인 사항

#### 로컬
- [ ] 데이터 로드 성공
- [ ] 차익거래 비용 계산 결과 표시
- [ ] 수익률 차트 표시
- [ ] 오류 없음

#### 클라우드
- [ ] 데이터 로드 성공 (로컬과 동일)
- [ ] 차익거래 비용 계산 결과 표시 (로컬과 동일)
- [ ] 수익률 차트 표시 (로컬과 동일)
- [ ] "no such table: upbit_daily" 오류 없음
- [ ] 오류 없음
