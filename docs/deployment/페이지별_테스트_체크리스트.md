## 1. 리스크 예측 대시보드

### 테스트 절차
1. 메뉴에서 "⚠️ 리스크 예측 대시보드" 선택
2. 사이드바에서 코인 선택 (BTC → ETH)
3. 날짜 선택 (최근 날짜)
4. "🔍 리스크 분석" 버튼 클릭

### 확인 사항

#### 로컬
- [ ] BTC 선택 시 데이터 로드 성공
- [ ] ETH 선택 시 데이터 로드 성공
- [ ] 리스크 점수 카드 표시 (고변동성 확률, 청산 리스크, 종합 리스크)
- [ ] 주요 지표 차트 표시 (고래 집중도, 펀딩비, OI 변화율)
- [ ] 동적 지표 섹션 표시 (하이브리드 모델 사용 시)
- [ ] 파생상품 지표 섹션 표시
- [ ] 거래소 유입/유출 섹션 표시
- [ ] 최근 리스크 이력 차트 표시

#### 클라우드
- [ ] BTC 선택 시 데이터 로드 성공 (로컬과 동일)
- [ ] ETH 선택 시 데이터 로드 성공 (로컬과 동일)
- [ ] 리스크 점수 카드 표시 (로컬과 동일)
- [ ] 주요 지표 차트 표시 (로컬과 동일)
- [ ] 동적 지표 섹션 표시 (로컬과 동일)
- [ ] 파생상품 지표 섹션 표시 (로컬과 동일)
- [ ] 거래소 유입/유출 섹션 표시 (로컬과 동일)
- [ ] 최근 리스크 이력 차트 표시 (로컬과 동일)
