## ⚠️ 주의사항

### 환경 변수 설정

Streamlit Cloud에서 다음 환경 변수를 설정해야 합니다:

- `SUPABASE_URL`: Supabase 프로젝트 URL
- `SUPABASE_KEY` 또는 `SUPABASE_SERVICE_ROLE_KEY`: Supabase API 키

### 데이터베이스

- 로컬 SQLite DB (`project.db`)는 2.3MB로 Streamlit Cloud 제한 내
- Supabase를 주 데이터 소스로 사용
- 모델 파일 9개 포함

### 알려진 제한사항

1. **날짜 범위**: 일부 기능은 2025-11-30 데이터가 없을 수 있음 (가장 가까운 날짜 사용)
2. **ETH 데이터**: 일부 ETH 파생상품 데이터가 부족할 수 있음
3. **고래 거래 데이터**: `transaction_direction`이 None인 경우 거래소 유입/유출이 0으로 표시될 수 있음
