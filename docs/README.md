# 문서 디렉토리 구조

이 디렉토리는 프로젝트의 모든 문서를 카테고리별로 정리한 것입니다.

## 📁 디렉토리 구조

```
docs/
├── deployment/          # 배포 관련 문서
│   ├── Streamlit Cloud 배포 가이드
│   ├── Docker 배포 가이드
│   └── 배포 체크리스트
│
├── guides/              # 사용 가이드 및 튜토리얼
│   ├── Trading Bot 가이드
│   ├── Transaction Direction 가이드
│   └── API 키 발급 가이드
│
├── reports/             # 프로젝트 보고서
│   ├── PM 업무지시서 및 진행보고서
│   ├── 서비스 리뷰 보고서
│   └── 완료 보고서
│
├── analysis/            # 데이터 분석 및 문제 해결 문서
│   ├── ETH 데이터 분석
│   ├── Project 2/3 분석
│   └── 데이터 수집 현황
│
└── README.md            # 이 파일
```

## 📚 주요 문서

### 배포 (deployment/)
- Streamlit Cloud 배포 가이드
- Docker 배포 가이드
- 배포 체크리스트 및 트러블슈팅

### 가이드 (guides/)
- Trading Bot 아키텍처 및 사용 가이드
- Transaction Direction 설정 가이드
- API 키 발급 및 설정 가이드

### 보고서 (reports/)
- PM 업무지시서 및 진행보고서
- 서비스 리뷰 보고서
- 프로젝트 완료 보고서

### 분석 (analysis/)
- ETH 데이터 수집 및 분석
- Project 2/3 데이터 품질 분석
- 데이터 수집 현황 및 문제 해결

## 🔍 문서 찾기

특정 주제의 문서를 찾으려면:
- **배포 관련**: `docs/deployment/`
- **사용 가이드**: `docs/guides/`
- **프로젝트 보고서**: `docs/reports/`
- **데이터 분석**: `docs/analysis/`
