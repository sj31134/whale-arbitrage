# 자동매매 봇

기존 프로젝트의 분석 결과를 활용한 데이터 기반 자동매매 시스템

## 주요 기능

1. **데이터 기반 시그널 생성**
   - 기존 RiskPredictor의 고변동성 확률 활용
   - 특성 중요도 기반 가중치 적용
   - 동적 변수 분석

2. **김치 프리미엄 필터**
   - 역프 또는 낮은 김프일 때만 매수 허용
   - 역프 상태일 때 포지션 크기 2배

3. **리스크 관리**
   - 손절매/익절 자동 실행
   - 최대 포지션 비율 제한

4. **텔레그램 알림**
   - 매수/매도 체결 알림
   - 에러 알림

## 설치 방법

```bash
# 1. 설치 스크립트 실행
python install_trading_bot.py

# 2. Streamlit 앱 실행
streamlit run app/main.py
```

## 사용 방법

1. Streamlit 앱에서 "🤖 자동매매 봇" 메뉴 선택
2. "설정" 탭에서 API 키 및 거래 파라미터 입력
3. "제어" 탭에서 봇 시작
4. "모니터링" 탭에서 실시간 상태 확인

## 주의사항

- **모의투자 환경에서 충분히 테스트 후 실거래 사용**
- API 키는 안전하게 보관하세요
- 설정 파일(`trading_bot/config/user_settings.json`)은 git에 커밋되지 않습니다

## 기존 시스템과의 관계

- **독립적인 모듈**: 기존 데이터 수집 DB와 분석 서비스에 영향 없음
- **읽기 전용**: 기존 `data/project.db`는 읽기만 하고 수정하지 않음
- **별도 설정**: `trading_bot/config/`에 별도 설정 파일 사용
