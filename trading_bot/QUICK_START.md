# 자동매매 봇 빠른 시작 가이드

## 1분 안에 시작하기

### 1단계: 설치
```bash
python install_trading_bot.py
```

### 2단계: 앱 실행
```bash
streamlit run app/main.py
```

### 3단계: 설정
1. 브라우저에서 "🤖 자동매매 봇" 메뉴 선택
2. "⚙️ 설정" 탭에서:
   - 업비트 API 키 입력
   - 거래 파라미터 설정
   - "💾 설정 저장" 클릭

### 4단계: 시작
1. "🎮 제어" 탭으로 이동
2. "▶️ 봇 시작" 버튼 클릭

## 필수 설정 항목

- ✅ 업비트 Access Key
- ✅ 업비트 Secret Key
- ⚪ 텔레그램 봇 토큰 (선택)
- ⚪ 텔레그램 채팅 ID (선택)

## 기본 거래 설정

- 대상 코인: BTC
- 초기 자본금: 1,000,000원
- 최대 포지션 비율: 30%
- 손절매: -5%
- 익절: +10%

## 주의사항

⚠️ **실거래 전 반드시 모의투자 환경에서 테스트하세요!**

## 문제 해결

### 봇이 시작되지 않을 때
- API 키 확인
- 설정 저장 확인
- `logs/trading_bot.log` 파일 확인
